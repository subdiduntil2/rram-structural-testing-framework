# rram-structural-testing-framework
This repository includes the codebase required to perform defect-oriented analysis &amp; test generation for RRAM-based Compute-in-memory (CIM) AI Accelerators. Currently, measurement and SPICE data availability (required for fault modeling analysis and RRAM device characteristics plotting) is not available publicly due to NDA limitations. The RRAM framework for workload mapping, synthetic image generation and fault modeling/masking analysis can run independently. 

Future steps include:
1. Full validation of the hybrid framework for the KWS dataset (current results are sub-optimal)
2. Algorithmic improvements, i.e. on loss function for synthetic input generation
3. Synthetic data for full functionality of the repository regarding SPICE fault modeling &amp RRAM measurement data analysis
